print()

print () er en funksjon som sender data (informasjon) fra kodefeltet til konsollen.

Input 
            print("Hei, verden!")
Output 
            Hei, verden!

range()

range () er en funksjon som genererer en sekvens av tall.

Eksempel på bruk av range () i en for-løkke:

Input 
            for x in range(5):
                print(x)
Output 
            0
            1
            2
            3
            4

len()

len () er en funksjon som returnerer lengden på et objekt.

Input 
            min_liste = [1, 2, 3, 4, 5]
            print(len(min_liste))
            min_string = "Hei"
Output 
            5
            3

type()

type () er en funksjon som returnerer typen til et objekt.

Input 
            min_liste = [1, 2, 3, 4, 5]
            print(type(min_liste))
            min_string = "Hei"
            print(type(min_string))
            mitt_tall = 3
            print(type(mitt_tall))
            mitt_desimaltall = 3.0
            print(type(mitt_desimaltall))
Output 
            <class 'list'>
            <class 'str'>
            <class 'int'>
            <class 'float'>

input()

input () er en funksjon som lar brukeren skrive inn data.

Input 
            navn = input("Skriv inn navnet ditt: ")
            print("Hei, " + navn)
Output 
            Skriv inn navnet ditt: Ola
            Hei, Ola
